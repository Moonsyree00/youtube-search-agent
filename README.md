# YouTube Search Agent

LLM-powered YouTube data collection agent for brand channel verification, video metadata collection, and CSV-based research workflows.

## Overview

YouTube Search Agent is a research-oriented data collection tool designed for non-technical research team members. It supports natural language task input, brand channel search, official account verification, CSV-based batch workflows, and local result export.

## Core Features

- Natural language task input
- YouTube brand channel search
- Official channel verification
- CSV batch workflows
- Video metadata collection by date range
- Local CSV export
- Desktop GUI prototype
